About the Server

Welcome to Creaking SMP — a crossplay Minecraft survival multiplayer server that supports both Java and Bedrock! Our goal is to create a chill, long-lasting experience with friendly players, vanilla gameplay, and a close-knit community.

Whether you're a builder, redstoner, explorer, or casual player, there's a place for you here. Our server is lightly moderated, grief-protected, and open 24/7 for all time zones.

How to Join

Java Edition:
IP: mc.creaking.fun
Version: 1.20+
Bedrock Edition:
Add the username: CreakingBedrock
Version: 1.20+

Make sure to join the Discord after joining the server for full access and community events!

Community Rules

  • 🌟 Be respectful to others
  • 🚫 No griefing or stealing
  • 🛡️ No hacks or cheats (x-ray, fly, etc.)
  • 👷 Build responsibly and don't lag the server
  • 📢 Keep chat friendly and clean

Failure to follow rules may result in a warning or ban.
